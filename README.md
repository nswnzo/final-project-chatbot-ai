# EduBot AI

EduBot AI adalah aplikasi chatbot berbasis web yang dibuat untuk membantu proses belajar dengan cara tanya jawab sederhana.

## Fitur
- Chatbot berbasis web
- Bisa menerima pertanyaan dari user
- Menampilkan jawaban secara langsung
- Tampilan sederhana dan mudah digunakan

## Teknologi
- Python
- Flask
- HTML
- CSS
- JavaScript
